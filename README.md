📋 Gestor de Tareas
Aplicación web desarrollada para la gestión de tareas personales, diseñada para organizar actividades de manera simple, rápida e intuitiva.
Este proyecto fue realizado como práctica de desarrollo Full Stack, con foco en el desarrollo backend, integración con base de datos y creación de interfaces dinámicas.

🚀 Funcionalidades


✅ Crear tareas


✏️ Editar tareas existentes


🗑️ Eliminar tareas


📌 Marcar tareas como completadas


🔍 Visualización dinámica de tareas


💾 Persistencia de datos


🎨 Interfaz responsive y minimalista


⚡ Aplicación rápida y ligera

⚙️ Instalación y ejecución

1️⃣ Clonar el repositorio
git clone https://github.com/eze2013/Gestor_de_tareas.git

2️⃣ Entrar en la carpeta del proyecto
cd Gestor_de_tareas

3️⃣ Crear entorno virtual (opcional)
Windows
python -m venv venvvenv\Scripts\activate
Linux / Mac
python3 -m venv venvsource venv/bin/activate

4️⃣ Instalar dependencias
pip install -r requirements.txt

5️⃣ Ejecutar la aplicación
python app.py

🌐 Acceso local
Abrir en el navegador:
http://127.0.0.1:5000
